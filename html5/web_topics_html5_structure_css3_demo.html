HTML5

The goal is to learn about the new HTML5 features of the language, as well as learn the basic use of JavaScript API with HTML5 markup.

The aim is get to know about the new semantic elements, and the basic form controls.

The goal is to get to know the most important HTML5's new features, such as Audio, Video, microdata, the new form elements, as well basics of the drawing to Canvas element.

CSS3

The goal is to learn CSS3 styles and practical application. There's also the introduction of CSS3 for use with the JavaScript programming language. The goal is to learn the basics of responsive design, allowing support for a variety of mobile device types.

Getting to know the new CSS selectors (CSS Selectors), CSS animations, transformations, and new text-style and font-style properties.

JavaScript

The goal is to learn the basics of JavaScript, the object-oriented programming in JavaScript, the updating DOM document object model, as well as event processing criteria.

There is also an introduction to JSON format, as well as to Ajax.

jQuery

jQuery library is used every modern JavaScript applications. It is the de facto standard that can be applied in all client-side Web development.

The goal is to learn the how to use jQuery library for a client-side Web application development. you will also getting to know how access to the basic features of the jQuery library, such as options, functions, and event handling, as well as to know the most important plug-in API (eg. jQueryUI).

HTML5 and JavaScript

We introduce to brand new HTML5 JavaScript API's. There is introduction to different JavaScript API's such as Video API, Offline Web Applications, storing of information for the local machine (Web Storage), Location API and Web Workers API.

The main issue addressed is the Canvas element to use JavaScript API for making programs.

JQuery Mobile

jQuery Mobile is a jQuery plug-in and widget-collection, which provides a platform-independent API for mobile application development and mobile device that allows a Web user experience is almost the workstation level.

The goal is to learn the jQuery Mobile library of the most important properties of a touch screen use, a small display resolution support, and page navigation.
