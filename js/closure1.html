Example 1 - How to use Closures in JavaScript

A closure is an inner function that has access to the outer (enclosing) function variables. You simple create a closure by adding a function inside another function. The closure has three scope chains accesses:

  • its own scope (variables defined in in curly brackets)
  • to outer function variables
  • to global (=window) variables. Inner function can also access to parameters of the outer function.
