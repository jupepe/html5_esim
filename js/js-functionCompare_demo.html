Example of How to use Sort arrays in JavaScript

In sorting function array.sort(),elements are sorted based on the relationship between each pair of elements "a" and "b" and the function's return values negative, zero or positive.

  • Less than 0: a is compared to be a lower index in the resulting array than b (a is before b)
  • Zero: a === b
  • Greater than 0: b is compared to be a lower index in the resulting array than a (b is before a)
